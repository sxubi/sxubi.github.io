### <center>Lecture 1. Sets</center>
**Definition** (Set)**.**  A *set* is a collection of objects.       
**Definition.** A statement is a sentence that can have a true value.        
**Definition.** The symbol $$\forall$$ means *for all*. The symbol $$\exists$$ means *there exists*. And $$\exists!$$ means "there exists a unique".    
**Definition** (Equality of sets)**.**  $$X$$ is equal to $$Y$$, written as $$X=Y$$ if $$\forall x, x\in X\Leftrightarrow x\in Y.$$     
**Definition** (Subset)**.**  $$X$$ is a *subset* of $$Y$$, written as $$X\subseteq Y$$ or $$X\subset Y$$, if all elements in $$X$$ are in $$Y$$, i.e. $$\forall x, x\in X\Rightarrow x\in Y$$.     
**Definition** (Intersection, union, set difference and power set). Given two sets $$X,Y$$,
* Intersection: $$X\cap Y=\{x:x\in X\text{ and }x\in Y\}$$
* Union: $$X\cup Y=\{x:x\in X\text{ or }x\in Y\}$$
* Set difference: $$X\backslash Y=\{x\in X:X\notin Y\}$$
* Power set: $$\mathscr{P}(x)=\{S:S\subseteq X\}$$

**Definition** (Ordered pair)**.**  An *order pair* $$(x,y)$$ is a pair of two items in which order matters. We have $$(x,y)=(x',y')$$ iff $$x=x',y=y'$$.       
**Definition** (Cartesian product)**.**  Given two sets $$X,Y$$, the *Cartesian product* of $$X$$ and $$Y$$ is $$X\times Y=\{(x,y):x\in X,y\in Y\}$$.      
**Definition** (Function)**.**  A *function* $$f:X\to Y$$ is a subset: $$\Gamma_f\subset X\times Y$$, called the *graph* of $$f$$, such that

$$\forall x\in X,\exists!y\in Y: (x,y)\in\Gamma_f.$$

This unique $$y$$ is denoted $$y=f(x)$$ and called the *value* of $$f$$ at $$x$$.     
**Terminology.** $$X$$ is the *domain* of $$f$$. $$Y$$ is the *codomain* of $$f$$. The *image* of $$X$$ under $$f$$ is the set

$$f(X)=\{y\in Y|\exists x\in X,f(x)=y\}.$$

### <center>Lecture 2. Functions and Relations</center>
**Definition** (Injective function)**.**  A function $$f:X\to Y$$ is *injective* if $$\forall x,y\in X, f(x)=f(y)\Rightarrow x=y.$$     
**Definition** (Surjective function)**.**  A function $$f:X\to Y$$ is *surjective* if $$\forall y\in Y, \exists x\in X,f(x)=y.$$     
**Definition** (Bijective function)**.**  A function is *bijective* if it is both injective and surjective.    
**Definition.** Let $$X,Y,Z$$ be sets and $$f:X\to Y, g:Y\to Z$$ be functions. Then the composition $$g\circ f$$ is the function $$g\circ f:X\to Z$$ such that $$\forall x\in X, (g\circ f)(x)=g(f(x))$$.    
**Definition.** Let $$X$$ be a set. The *identity function* is the function

$$\text{id}_X:X\to X\text{ such that }\forall x\in X, \text{id}_X(x)=x.$$

**Axiom** (Axiom of choice)**.**  Given a family of non-empty sets $$X_i,i\in I$$, there exists a choice function $$f:I\to U_{i\in I}X_i$$ such that $$f(i)\in X_i$$.     
**Definition.** A *relation* $$\mathcal{R}$$ on a set is a subset $$R\subseteq X\times X.$$ We write $$x\sim_\mathcal{R}y\Leftrightarrow (x,y)\in\mathcal{R}.$$    
**Definition** (Properties of relations)**.**  Let $$\mathcal{R}$$ be a relation on $$X$$.    
&emsp;1. It is *reflexive* if $$\forall x\in X,x\sim_\mathcal{R}x$$.     
&emsp;2. It is *symmetric* if $$\forall x,y\in X, x\sim_\mathcal{R}y\Rightarrow y\sim_\mathcal{R}x$$.     
&emsp;3. It is *transitive* if $$\forall x,y,z\in X,x\sim_\mathcal{R} y,y\sim_\mathcal{R}z\Rightarrow x\sim_\mathcal{R}z$$.     
**Definition** (Equivalence relation)**.**  A relation is an *equivalence relation* if it is reflexive, symmetric and transitive.    
**Definition** (Equivalence class)**.**  If $$\mathcal{R}$$ is an equivalence relation, then the *equivalence class* $$[x]$$ is the set of all elements that are related via $$\sim_\mathcal{R}$$ to $$x$$:

$$[x]=\{y\in X|(x,y)\in\mathcal{R}\}.$$

**Definition** (Quotient set)**.**  If $$\mathcal{R}$$ is an equivalence relation on $$X$$, then we denote the *quotient set*:

$$X/\mathcal{R}=\{[x]|x\in X\}.$$

Quotient set is the set of equivalence classes.    
**Definition** (Partition of a set)**.**  A *partition* of a set $$X$$ is a collection of subsets $$X_i$$ of $$X$$ such that each element of $$X$$ is in exactly one of $$X_i$$.    
**Theorem.** To give an equivalence relation on $$X$$ is the same as to give a partition of $$X$$.

### <center>Lecture 3. Division and Modulo</center>
**Definition** (Factor of integers)**.**  Given $$a,b\in\mathbb{Z}$$, we say $$a$$ *divides* $$b$$, $$a$$ is a *factor* of $$b$$, or $$a|b$$ if $$\exists c\in\mathbb{Z}, b=ac$$. For any $$b$$, $$\pm 1$$ and $$\pm b$$ are always factors of $$b$$. The other factors are called *proper factors*.       
**Theorem** (Division Algorithm). Given $$a,b\in\mathbb{N}, b\neq 0$$, there are unique $$q,r\in\mathbb{Z}$$ with $$a=qb+r,0\leq r<b.$$     
**Definition** (Common factor of integers)**.**  A *common factor* of $$a$$ and $$b$$ is a number $$c\in\mathbb{Z}$$ such that $$c|a$$ and $$c|b$$.    
**Definition** (Greatest common divisor)**.**  The *greatest common divisor* $$\gcd(a,b)$$ of two numbers $$a,b\in\mathbb{N}$$ is a number $$d\in\mathbb{N}$$ such that     
&emsp;(1) $$d$$ is a common factor of $$a$$ and $$b$$, and    
&emsp;(2) if $$c$$ is also a common factor, $$c|d$$.    
**Proposition.** If $$c|a$$ and $$c|b$$, then $$c|(ua+vb)$$ for all $$u,v\in\mathbb{Z}$$.    
**Theorem.** Let $$a,b\in\mathbb{N}$$, then $$\gcd(a,b)$$ exists.    
**Corollary.** Let $$d=\gcd(a,b),$$ then $$d$$ is the smallest positive linear combination of $$a$$ and $$b$$.    
**Corollary.** (BÃ©zout's identity)**.**  Let $$a,b\in\mathbb{N}$$ and $$c\in\mathbb{Z}$$. Then there exists $$u,v\in\mathbb{Z}$$ with $$c=ua+vb$$ iff $$\gcd(a,b)|c$$.     

> **Relatively prime (Coprime).** If $$a,b$$ are comprime, then $$\text{gcd}(a,b)=1$$. And if $$c$$ divides $$a$$, then $$c,b$$ are coprime.

**Proposition** (Euclid's Algorithm)**.**  If we continuously break down $$a$$ and $$b$$ by the following procedure:

$$\begin{align*}
a&=q_1b+r_1,\\
b&=q_2r_1+r_2,\\
r_1&=q_3r_2+r_3,\\
&\vdots\\
r_{n-2}&=q_nr_{n-1}.
\end{align*}$$

Then we have $$\gcd(a,b)=r_{n-1}$$.    
**Definition** (Modulo). If $$a,b\in\mathbb{Z}$$ have the same remainder after division by $$m$$, i.e. $$m|(a-b)$$, we say $$a$$ and $$b$$ are *congruent modulo* $$m$$, and write

$$a\equiv b\pmod m.$$

**Proposition.** If $$a\equiv b\pmod m$$, and $$d|m$$, then $$a\equiv b\pmod d$$.    
With $$m$$ fixed, $$a\equiv b\pmod m$$ is an equivalence class. The set of equivalence classes is written as $$\mathbb{Z}/(m\mathbb{Z})$$.    
**Example.** $$\mathbb{Z}_3=\{[0],[1],[2]\}.$$      
**Proposition.** If $$a\equiv b\pmod m$$ and $$u\equiv v\pmod m$$, then $$a+u\equiv b+v\pmod m$$ and $$au\equiv bv\pmod m.$$

### <center>Lecture 4. Group and Subgroup</center>
**Definition** (Binary operation)**.** A *binary operation* on a set $$S$$ is a way of combining two elements to get a new element, which is a map $$*:S\times S\to S.$$    
**Definition** (Group)**.** A *group* is a set $$G$$ with a binary operation $$*$$ satisfying the following axioms:    
&emsp;1. (Identity) There is some $$e\in G$$ such that for all $$a$$, $$a*e=e*a=a.$$     
&emsp;2. (Inverse) For all $$a\in G$$, there is some $$a^{-1}\in G$$ such that $$a*a^{-1}=a^{-1}*a=e$$.    
&emsp;3. (Associativity) For all $$a,b,c,\in G$$, we have $$(a*b)*c=a*(b*c)$$.    
**Example.** Groups: $$(\mathbb{Z},+),(\mathbb{Q},+),(\mathbb{Q}\backslash\{0\},\times),(\mathbb{R}\backslash\{0\},\times),(\mathbb{C}\backslash\{0\},\times),(\{1,-1,i,-i\},\times).$$     
**Definition** (Order of a group)**.** The *order* of the group, denoted by $$|G|$$, is the number of elements in $$G$$. A group is a finite group if the order is finite.    
**Definition** (Abelian group)**.** A group is *abelian* if $$\forall a,b\in G, a*b=b*a$$ (commutativity).     
**Example.** The following are abelian groups: $$(\mathbb{Z},+),(\mathbb{Q},+),(Z/n,+_n),(Q^*,\times),(\{-1,1\},\times).$$    
**Proposition.** Let $$(G,*)$$ be a group. Then    
&emsp; 1. The identity is unique.    
&emsp; 2. Inverses are unique.     
**Proposition.** Let $$(G,*)$$ be a group and $$a,b\in G$$. Then    
&emsp; 1. $$(a^{-1})^{-1}=a.$$    
&emsp; 2. $$(ab)^{-1}=b^{-1}a^{-1}.$$    
**Definition** (Subgroup)**.** A $$H$$ is a *subgroup* of $$G$$, written $$H\leq G$$, if $$H\subseteq G$$ and $$H$$ with the restricted operation $$*$$ from $$G$$ is also a group.    
**Lemma** (Subgroup criteria I)**.** Let $$(G,*)$$ be a group and $$H\subseteq G$$. $$H\leq G$$ iff      
&emsp; 1. $$e\in H$$.     
&emsp; 2. $$\forall a,b\in H,ab\in H$$.     
&emsp; 3. $$\forall a\in H, a^{-1}\in H$$.    
**Lemma** (Subgroup criteria II)**.** A subset $$H$$ is a subgroup of $$G$$ iff   
&emsp; 1. $$H$$ is non-empty;    
&emsp; 2. $$\forall a,b\in H,ab^{-1}\in H$$.    

> The intersection of 2 subgroups is also a subgroup.      
> Suppose $$U$$ is the subset of a group $$G$$. $$S=\{H\leq G: U\subset H\}, \hat U=\bigcap_{H\subset S}H$$ is the smallest subgroup containing $$U$$. If $$\forall g\in G, u\in U, gug^{-1}\in U$$, then $$\hat U\triangleleft G$$. Specifically, if $$U=\{xyx^{-1}y^{-1}:x,y\in G\}$$, then $$G'=\hat U$$ is called the *commutator subgroup* of $$G$$. $$G'\triangleleft G$$ and $$G/G'$$ is abelian.

### <center>Lecture 5. Homomorphism and Isomorphism</center>
**Proposition.** The subgroups of $$(\mathbb{Z},+)$$ are exactly $$n\mathbb{Z}$$, for $$n\in\mathbb{N}\cup\{0\}.$$    
**Definition** (Group homomorphism)**.** Let $$(G,*)$$ and $$(H,\times)$$ be groups. A function $$f:G\to H$$ is a group homomorphism if 

$$\forall g_1,g_2\in G, f(g_1*g_2)=f(g_1)\times f(g_2).$$    

**Definition** (Group isomorphism)**.** *Isomorphisms* are bijective homomorphisms. Two groups are *isomorhpic* if there exists an isomorphism between them. We write $$G\cong H$$.    
**Proposition.** Suppose $$f:G\to H$$ is a homomorphism. Then    
&emsp; 1. Homomorphisms send the identity to the identity: $$f(e_G)=e_H$$.    
&emsp; 2. Homomorphisms send inverses to inverses: $$f(a^{-1})=f(a)^{-1}.$$    
&emsp; 3. The composite of 2 group homomorphisms is a group homomorphism.    
&emsp; 4. The inverse of an isomorphism is an isomorphism.    
**Definition** (Image of homomorphism)**.** If $$f:G\to H$$ is a homomorphism, then the *image* of $$f$$ is 

$$\text{im} f=f(G)=\{f(g):g\in G\}.$$

**Definition** (Kernel of homomorphism)**.** The *kernel* of $$f$$ is written as

$$\ker f=f^{-1}(\{e_H\})=\{g\in G: f(g)=e_H\}.$$

**Proposition.** Both the image and kernel are subgroups of the respective groups: $$\text{im }f\leq H,\ker f\leq G.$$    

> *Proof.* Note that the image and kernel are not empty. Suppose $$b_1,b_2\in\text{Im }f$$. Now $$\exists a_1,a_2\in G$$ such that $$f(a_i)=b$$. Then $$b_1b_2^{-1}=f(a_1)f(a_2^{-1})=f(a_1a_2^{-1})\in\text{Im }f.$$     
> Suppose $$b_1,b_2\in\ker f$$. We have $$f(b_1b_2^{-1})=f(b_1)f(b_2)^{-1}=e.$$ 

**Proposition.** Given any homomorphismm $$f:G\to H$$ and any $$a\in G$$, for all $$k\in\ker f$$, $$aka^{-1}\in\ker f.$$     

> *Proof.* $$f(aka^{-1})=f(a)f(k)f(a)^{-1}=f(a)f(a)^{-1}=e.$$

**Proposition.** For all homomorphisms $$f:G\to H$$, $$f$$ is    
&emsp; 1. *surjective* iff $$\text{im}f=H$$.    
&emsp; 2. *injective* iff $$\ker f = \{e\}$$. 

> *Proof.* 2. If $$f$$ is injective, then by definition $$\ker f=\{e\}$$. If $$\ker f=\{e\}$$, then given any $$a,b$$ such that $$f(a)=f(b)$$, $$f(ab^{-1})=f(a)f(b)^{-1}=e.$$ Thus $$ab^{-1}\in\ker f=\{e\}$$. Thus $$a=b$$. 

### <center>Lecture 6. Cyclic Groups and Dihedral Groups</center>
#### Cyclic Groups
A *cyclic group* is the group of the form $$\{e,a,a^2,a^3,\cdots,a^{n-1}\}$$, where $$a^n=e$$.    
**Definition** (Cyclic group $$C_n$$)**.** A group $$G$$ is *cyclic* if

$$\exists a\forall b,\exists n\in\mathbb{Z},\quad b=a^n,$$

i.e. every element is some power of $$a$$. Such an $$a$$ is called a *generator* of $$G$$. We write $$C_n$$ for the cyclic group of order $$n$$.     
**Example.** (1) $$\mathbb{Z}$$ is cyclic with generator $$1$$ or $$-1$$. It is the infinite cyclic group. (2) $$(\{+1,-1\},\times)$$ is cyclic with generator $$-1$$. (3) $$(\mathbb{Z}_n,+)$$ is cyclic with all numbers comprime with $$n$$ as generators.     
Given a group $$G$$ and $$a\in G$$, we write $$\langle a \rangle$$ for the cyclic group generated by $$a$$, i.e. the subgroup of all powers of $$a$$. It is the smallest subgroup containing $$a$$.   
**Definition** (Order of element)**.** The *order* of an element $$a$$ is the smallest integer $$n$$ such that $$a^n=e$$. If $$n$$ doesn't exist, $$a$$ has infinite order. Write $$\text{ord}(a)$$ for the order of $$a$$.     
**Lemma.** For $$a$$ in $$G$$, $$\text{ord}(a)=|\langle a\rangle|$$.    

> If $$x^n=e$$, then the order of $$x$$ divides $$n$$.

**Proposition.** Cyclic groups are abelian.      
**Definition** (Exponent of group)**.** The *exponent* of a group $$G$$ is the smallest positive integer $$n$$ such that $$a^n=e$$ for all $$a\in G$$.     

#### Dihedral Groups
**Definition.** Let $$S$$ be a subset of $$\mathbb{R}^n$$. Let $$A\in GL_n(\mathbb{R})$$. Write $$AS=\{Av:v\in S\}$$. If $$AS=S$$, we say $$A$$ *preserves* $$S$$.    
**Proposition.** For any $$S\subseteq\mathbb{R}^n$$, the set $$H=\{A\in GL_n(\mathbb{R}):AS=S\}$$ is a subgroup of $$L_n(\mathbb{R}$$, called the symmetry group of $$S$$.    
**Definition.** Let $$n>2$$. Let $$P_n$$ be the regular $$n$$-gon in $$\mathbb{R}^2$$ with vertices $$\displaystyle\begin{pmatrix}\cos 2\pi k/n\\\sin 2\pi k/n\end{pmatrix}$$, $$k=0,\cdots, n-1$$. The *dihedral group* $$D_{2n}$$ is the symmetry group of $$P_n$$, $$D_{2n}=\{A\in GL_2(\mathbb{R}):AP_n=P_n.\}$$      
**Proposition.** The group $$D_{2n}$$ has order $$2n$$.    
**Corollary.** Every element of $$D_{2n}$$ is either a rotation or a reflection.    
$$D_{2n}$$ is a group generated by $$r$$ and $$s$$, and every element can be written as a product of $$r$$'s and $$s$$'s. Whenever we see $$r^n$$ and $$s^2$$, replace it by $$e$$. Whenever we see $$srs^{-1}$$, replace it by $$r^{-1}$$. Formally, $$D_{2n}$$ can be written as

$$\begin{align*}
D_{2n} &=\langle r,s|r^n=s^2=e,srs^{-1}=r^{-1}\rangle\\
&=\{e,r,r^2,\cdots,r^{n-1},s,rs,r^2s,\cdots,r^{n-1}s\}.
\end{align*}$$

> An element in the form of $$r^k s$$ represents a reflection. Thus its order is $$2$$.

> **Subgroups of** $$D_{2n}$$    
> The number of subgroups in $$D_{2n}$$ is $$\tau(n)+\sigma(n)$$, where $$\tau(n)$$ is the number of divisors of $$n$$ and $$\sigma(n)$$ is the sum of divisors of $$n$$. Every subgroup of $$D_{2n}$$ is either cyclic or dihedral. A complete listing:    
> &emsp;1. $$\langle r^d\rangle, d|n$$,    
> &emsp;2. $$\langle r^d,r^is\rangle,d|n,0\leq i\leq d-1$$.     
> If $$n$$ is odd, the normal subgroups are given by $$D_{2n}$$ and all $$\langle r^d\rangle$$. If $$n$$ is even, there are two more normal subgroups: $$\langle r^2,s\rangle$$ and $$\langle r^2,rs\rangle.$$

### <center>Lecture 7. Symetric Group and Permutation</center>
**Definition** (Permutation)**.** A *permutation* of $$X$$ is a bijection from a set $$X$$ to $$X$$ itself. The set of all permutations on $$X$$ is $$\text{Sym}X$$.    
**Theorem.** $$\text{Sym}X$$ with composition forms a group.    
**Definition** (Symmetric group $$S_n$$)**.** If $$X$$ is finite, say $$|X|=n$$, we write $$\text{Sym}X=S_n$$. This is the *symmetric group* of degree $$n$$.    
The order of $$S_n$$ is $$n!$$.    
**Example** (Two row notation)**.** If $$\sigma:X\to X$$, we write $$1,2,3,\cdots, n$$ on the top line and their images below

$$\begin{pmatrix}
1 & 2 & 3 & \cdots & n\\
\sigma(1) &\sigma(2) &\sigma(3)&\cdots&\sigma(n)
\end{pmatrix}$$

**Example.**     
1. When $$n=1, S_1=\left\{\begin{pmatrix}1\\1\end{pmatrix}\right\}=\{e\}\cong C_1.$$    
2. When $$n=2, S_2=\left\{\begin{pmatrix}1&2\\1&2\end{pmatrix}\begin{pmatrix}1&2\\2&1\end{pmatrix}\right\}\cong C_2.$$    
3. When $$n=3$$, 

$$S_3=\left\{\begin{array}{l}
\begin{pmatrix}1&2&3\\1&2&3\end{pmatrix},
\begin{pmatrix}1&2&3\\2&3&1\end{pmatrix},
\begin{pmatrix}1&2&3\\3&1&2\end{pmatrix}\\
\begin{pmatrix}1&2&3\\2&1&3\end{pmatrix},
\begin{pmatrix}1&2&3\\3&2&1\end{pmatrix},
\begin{pmatrix}1&2&3\\1&3&2\end{pmatrix}
\end{array}\right\}\cong D_6.$$

$$S_n$$ is not abelian for $$n\geq 3$$. $$D_{2n}$$ can be viewed as a subgroup of $$S_n$$.     

> **Alternating group** $$\mathbf{A_3}=\{e,(123),(132)\}$$.      
> **Symmetric group** $$\mathbf{S_4}$$    
> Number of subgroups: 30. Including $$A_3,A_4$$, $$V_4=\{e,(12)(34),(13)(24),(14)(23)\}$$ (order $$4$$), $$\langle (12)(34)\rangle$$ (order $$4$$), $$D_8$$.     
> Number of normal subgroups: 4. Including $$A_4,V_4$$.        
> **Alternating group** $$\mathbf{A_4} = \langle(123),(12)(34)\rangle$$     
> Number of subgroups: 10. Including $$V_4,A_3$$.     
> Number of normal subgroups: 3. Including $$V_4$$.    

**Example** (Cycle notation)**.** If a map sends $$1\mapsto 2,2\mapsto 3,3\mapsto 1$$, then we write it as a cycle $$(123)$$. Usually the smallest number is written first and numbers not moving are left out. The inverse is the cycle written other way round, e.g. $$(123)^{-1}=(321)=(132)$$.    
**Example.** Simplifying $$(123)(12)$$: $$1\mapsto 2\mapsto 3, 2\mapsto 1\mapsto 2,3\mapsto1$$, thus $$(123)(12)=(13)(2).$$    
**Definition** ($$k$$-cycles and transpositions)**.** We call $$(a_1a_2a_3\cdots a_k)$$ a $$k$$-*cycle*. $$2$$-cycles are called *transpositions*. Two cycles are *disjoint* if no number appears in both cycles.    
**Lemma.** Disjoint cycles commute.     
**Theorem.** Any permutation in $$S_n$$ can be written essentially uniquely as a product of disjoint cycles.     
**Definition** (Cycle type)**.** Write a permutation $$\sigma\in S_n$$ in disjoint cycle notation. The *cycle type* is the list of cycle lengths.    
**Example.** $$(12)$$ has cycle type 2. $$(12)(34)$$ has cycle type $$2,2$$.     
**Lemma.** For $$\sigma\in S_n$$, the order of $$\sigma$$ is the least common multiple of cycle lengths in the disjoint cycle notation. In particular, a $$k$$-cycle has order $$k$$.    

> Let $$G(n)$$ denote the greatest order of an element in $$S_n$$. To make $$G(x)$$ maximized, choose prime powers whose sum is $$\leq n$$ and whose product is as large as possible. A prime power is a positive integer which is a positive integer power of a single prime number.
> The symmetry group $$S_n$$ can be generated by 1. $$\{(jk)\}, 1\leq j<k\neq n$$; 2. $$\{(j\,j+1)\},1\leq j\leq n-1$$; 3. $$\{(1k)\},1<k\leq n$$; 4. $$\{(12),(12\cdots n)\}$$.


### <center>Lecture 8. Permutations</center>
#### *Permutation*
**Proposition.** Every permutation is a product of transpositions.      

> We have $$(a_1a_2\cdots a_k) = (a_1a_2)(a_2a_3)\cdots(a_{k-1}a_k)$$.
> Also $$(ab)=(1a)(1b)(1a).$$

**Theorem.** Writing $$\sigma\in S_n$$ as a product of transpositions in different way, $$\sigma$$ is either always composed of an even number of transpositions, or always an odd number of transpositions.    
**Definition** (Sign of permutation). Viewing $$\sigma\in S_n$$ as a product of transpositions, $$\sigma =\tau_1\cdots\tau_l$$, we call $$\text{sgn}(\sigma)=1$$, we call $$\sigma$$ an even permutation. If $$\text{sgn}(\sigma)=-1$$, we call $$\sigma$$ an odd permutation.    
**Theorem.** For $$n\geq 2$$, $$\text{sgn}:S_n\to\{\pm 1\}$$ is a surjective group homomorphism.    
Note that if $$\sigma=\sigma_1\sigma_2\cdots \sigma_l$$, then $$\text{sgn}(\sigma)=\text{sgn}(\sigma_1)\cdots\text{sgn}(\sigma_l).$$ If a cycle $$\sigma$$ has length $$k$$, then $$\text{sgn}(\sigma)=(-1)^{k-1}.$$    
**Lemma.** $$\sigma$$ is an even permutation iff the number of cycles of even length is even.    
**Definition** (Alternating group $$A_n$$)**.** The *alternating group* $$A_n$$ is the kernel of $$\text{sgn}$$, i.e. the even permutations. Since $$A_n$$ is a kernel of a group homomorphism, $$A_n\leq S_n$$.    
> **Theorem.** Let $$n\geq 3,n\in\mathbb{N}.$$ Then the alternating group $$A_n$$ is generated by the set of 3-cycles: $$\{(1in):1<i\leq n-1\}.$$
> **Theorem.** The alternating groups $$A_n$$ are simple for $$n\geq 5$$.

**Proposition.** Any subgroup of $$S_n$$ contains either no odd permutations or exactly half.    

### <center>Lecture 9. Lagrange's Theorem, Fermat-Euler Theorem and Direct Product</center>
#### Lagrange's Theorem 
**Definition.** (Cosets)**.** Let $$H\leq G$$ and $$a\in G$$. The set $$aH=\{ah:h\in H\}$$ is a *left coset* of $$H$$ and $$Ha=\{ha:h\in H\}$$ is a right coset of $$H.$$     
**Proposition.** $$aH=bH\Leftrightarrow b^{-1}a\in H$$.   

> *Proof.* $$(\Rightarrow)$$ Since $$a\in aH,a\in bH$$, then $$a=bh$$ for some $$h\in H$$. Thus $$b^{-1}a=h\in H$$. $$(\Rightarrow)$$ Let $$b^{-1}a=h_0$$. Then $$a=bh_0$$. $$\forall ah\in aH,$$ we have $$ah=b(h_0h)\in bH$$. Thus $$aH\subseteq bH$$. Similarly, $$bH\subseteq aH$$. So $$aH=bH$$. 

> $$kH=H\Leftrightarrow k\in H$$. This is an important connection between $$kH$$ and $$k\in H$$. May consider to construct mapping and determine the sujectivity and injectivity.    
> If $$H\subseteq G$$ and $$k,a\in G$$, then $$kaH=aH\Leftrightarrow k\in aH$$. 

**Lemma.** Given a group $$G$$ and a subgroup $$H$$, define the equivalence relation on $$G$$ with $$a\sim b$$ iff $$b^{-1}a\in H$$. The equivalence classes are the left cosets of $$H$$.     

> *Proof.* 1. Reflexivity: $$aa^{-1}=e\in H, a\sim a$$. 2. Symmetry: $$a\sim b\Rightarrow b^{-1}a\in H\Rightarrow a^{-1}b\in H\Rightarrow b\sim a.$$ 3. Transitivity: If $$a\sim b,b\sim c$$, then $$b^{-1}a,c^{-1}b\in H$$. Thus $$c^{-1}bb^{-1}a=c^{-1}a\in H\Rightarrow a\sim c$$.

**Lemma.** The left cosets of a subgroup $$H\leq G$$ partition $$G$$, and every coset has the same size.    

> *Proof.* For each $$a\in G, a\in aH$$. Thus the union of all cosets gives all of $$G$$. Now we need to show for all $$a,b\in G$$, the cosets $$aH$$ and $$bH$$ are either the same or disjoint.     
> Suppose $$aH,bH$$ are not disjoint. Let $$ah_1=bh_2\in aH\cap bH$$. Then $$b^{-1}a=h_2h_1^{-1}\in H$$. Thus $$aH=bH$$. Note that $$f:H\to aH, f(h)=ah$$ is invertible with inverse $$f^{-1}(h)=a^{-1}h$$. Thus there is a bijection between $$H$$ and $$aH$$ and they have the same size.

This lemma is the actual proof of the Lagrange's theorem.          
**Definition** (Index of a subgroup)**.** The *index* of $$H$$ in $$G$$, written $$|G:H|$$, is the number of left cosets of $$H$$ in $$G$$.    
**Theorem** (Lagrange's theorem)**.** If $$G$$ is a finite group and $$H$$ is a subgroup of $$G$$, then $$|H|$$ divides $$|G|$$. Particularly,

$$|H||G:H|=|G|.$$

The converse is not true.    
**Corollary.** The order of an element divides the order of the group, i.e. for any finite group $$G$$ and $$a\in G$$, $$\text{ord}(a)$$ divides $$|G|$$.      
**Corollary.** The exponent of a group divides the order of the group, i.e. for any finite group $$G$$ and $$a\in G$$, $$a^{|G|}=e$$.       
**Corollary.** Group of prime order are cyclic and generated by every non-identity element.    
**Example.** Consider $$(\mathbb{Z},+)$$ and for fixed $$n$$, take the subgroup $$n\mathbb{Z}$$. The cosets are $$0+H, 1+H,\cdots, (n-1)+H$$. Write these as $$[0],[1],[2],\cdots,[n-1]$$. To perform arithmetic mod $$n$$, define $$[a]+[b]=[a+b]$$, and $$[a][b]=[ab]$$ (they are well-defined). $$(\mathbb{Z}_n,+_n)$$ is a group. For multiplication, we can only take elements which have inverses. Call the set of them $$U_n=\{[a]:\text{gcd}(a,n)=1\}$$.    
**Definition** (Euler totinent function)**.** $$\phi(n)=|U_n|$$.    
**Example.** If $$p$$ is a prime, $$\phi(n)=p-1,\phi(4)=2$$.    
**Proposition.** $$U_n$$ is a group under multiplication $$\mod n$$.    
**Theorem** (Fermat-Euler theorem)**.** Let $$n\in\mathbb{N}$$ and $$a\in\mathbb{Z}$$ coprime to $$n$$. Then

$$a^{\phi(n)}\equiv 1\pmod n.$$

In particular, (Fermat's Little Theorem) if $$n=p$$ is a prime, then for any $$a$$ not a mulptiple of $$p$$, 

$$a^{p-1}\equiv 1\pmod p.$$

#### Direct Product
**Definition** (Direct product of groups)**.** Given two groups $$(G,\circ)$$ and $$(H,\bullet)$$, define a set $$G\times H=\{(g,h):g\in G, h\in H\}$$ and an operation $$(a_1,a_2)*(b_1,b_2)=(a_1\circ b_1,a_2\bullet b_2)$$. This forms a group.    
**Proposition.** $$C_n\times C_m\cong C_{nm}$$ iff $$\text{gcd}(m,n)=1$$.    

> *Proof.* Suppose $$\text{gcd}(m,n)=1.$$ Let $$C_n=\langle a\rangle, C_m=\langle b\rangle$$. Let $$k$$ be the order of $$(a,b)^k=(a^k,b^k)=e$$. This is possible only if $$n$$, $$m$$ divide $$k$$, $$k$$ is a common multiple of $$m,n$$. Thus $$k=\text{lcm}=mn.$$ Consider $$\langle(a,b)\rangle\leq C_n\times C_m,$$ which has $$mn$$ elements. $$C_n\times C_m$$ also has $$mn$$ elements, thus $$C_n\times C_m\cong \langle(a,b)\rangle\cong C_{nm}.$$

**Proposition** (Direct product theorem)**.** Let $$H_1,H_2\leq G$$. Suppose the following are true:   
&emsp;1. $$H_1\cap H_2=\{e\}$$.    
&emsp;2. $$\forall a_i\in H_i, a_1a_2=a_2a_1$$.    
&emsp;3. $$\forall a\in G,\exists a_i\in H_i, a=a_1a_2$$. Also write this as $$G=H_1H_2$$.    
Then $$G\cong H_1\times H_2$$.
    
> The order of a direct group $$G\times H$$ is equals to $$|G||H|$$.       
> The order of an element $$(g,h)$$ is the least common multiplier of the orders of $$g$$ and $$h$$: $$|(g,h)|=\text{lcm}(|g|,|h|).$$ This could be used to determine whether two direct product groups are isomorphic by checking their maximum order of elements.

### <center>Lecture 10. Quotient Group and Nomality</center>
**Definition** (Nomral subgroup)**.** A subgroup $$K$$ of $$G$$ is *a normal subgroup* if

$$\forall a\in G,\forall k\in K,\quad aka^{-1}\in K.$$

We write $$K\triangleleft G$$. This is equivalent to:    
&emsp; 1. $$\forall a\in G, aK=Ka$$, i.e. left cosets equal to right coset.    
&emsp; 2. $$\forall a\in G, aKa^{-1}=K.$$    
**Lemma.** 1. Every subgroup of index 2 is normal.     
&emsp; 2. Any subgroup of an abelian group is normal.    

> *Proof.* 1. If $$K\leq G$$ has index 2, there are two possible cosets $$K$$ and $$G\backslash K$$. As $$eK=Ke$$ and cosets partition $$G$$, the other left coset and right coset must be $$G\backslash K.$$ So all left cosets and right cosets are the same.     
> 2. Note that $$\forall a\in G, k\in K, aka^{-1}=k\in K$$.


> If $$H\leq G$$, then $$\forall g\in G, gHg^{-1}\leq G$$. Also it has the same number of elements as $$H$$ (the map $$h\mapsto ghg^{-1}$$ is bijiective).    
> If $$H$$ is the only subgroup with order $$d$$, then $$H\triangleleft G$$.     
> If $$N$$ is a normal subgroup with index $$m$$, then $$\forall g\in G, (gN)^m=N$$. The reason is that the order of $$gN$$ divides the order of the quotient group.

**Proposition.** Every kernel is a normal subgroup.    

> *Proof.* Given homomorphism $$f:G\to H$$ and some $$a\in G$$, for all $$k\in\ker f$$, we have $$f(aka^{-1})=f(a)ef(a)^{-1}=e$$. Thus $$aka^{-1}\in\ker f$$. 

**Proposition.** Let $$K\triangleleft G$$. Then the set of (left) cosets of $$K$$ in $$G$$ is a group under the operation $$aK*bK=(ab)K.$$    
&emsp; 1. Closure: If $$aK,bK$$ are cosets, then $$(ab)K$$ is also a coset.    
&emsp; 2. Identity: The identity is $$eK=K$$.    
&emsp; 3. Inverse: The inverse of $$aK$$ is $$a^{-1}K$$.    
&emsp; 4. Associativity: Follows the associativity of $$G$$.    

> *Proof.* We first show that the operation is well-defined. If $$aK=a'K, bK=b'K$$, we need to show that $$aK*bK=a'K*b'K$$. We know that $$a'=ak_1,b'=bk_2$$ for some $$k_1,k_2\in K$$. Then $$a'b'=ak_1bk_2$$. We know that $$b^{-1}k_1b=k_3\in K$$. Thus $$a'b'=abk_3k_2\in (ab)K$$. Thus picking a different representative of the coset gives the same product.

**Definition** (Quotient group)**.** Given a group $$G$$ and a normal subgroup $$K$$, the *quotient group* or *factor group* of $$G$$ by $$K$$, written as $$G/K$$, is the set of (left) cosets of $$K$$ in $$G$$ under the operation $$aK*bK=(ab)K$$.    

> If $$H\triangleleft G, K \triangleleft G$$, then $$HK\triangleleft G$$.

The set of left cosets also exists for non-normal subgroups, but the group operation is not well-defined.    
**Example.** 1. Take $$G=\mathbb{Z}$$ and $$H=n\mathbb{Z}$$. $$H$$ must be normal since $$G$$ is abelian. The cosets are $$k+n\mathbb{Z}$$ for $$0\leq k<n$$. The quotient group is $$\mathbb{Z}_n$$. We can write $$\mathbb{Z}/(n\mathbb{Z})=\mathbb{Z}_n$$. These are the only subgroups.    
&emsp; 2. Take $$K=\langle r\rangle\triangleleft D_6$$. We have two cosets $$K$$ and $$sK$$. So $$D_6/K$$ has order 2 and is isomorphic to $$C_2$$.    
&emsp; 3. Take $$K=\langle r^2\rangle\triangleleft D_8.$$ $$G/K$$ should have 4 elements. We have $$G/K=\{K,rK=r^3K,sK=sr^2K, srK=sr^3K\}$$. All elments except $$K$$ has order 2, so $$G/K\cong C_2\times C_2$$.    
**Lemma.** Given $$K\triangleleft G$$, the quotient map $$q:G\to G/K$$ with $$g\mapsto gK$$ is a surjective group homomorphism.    

> *Proof.* $$q(ab)=(ab)K=aK*bK=q(a)q(b)$$. Thus $$q$$ is a group homomorphism. Also for all $$aK\in G/K, q(a)=aK$$. Thus it's surjective.  

**Proposition.** The quotient of a cyclic group is cyclic.    

> *Proof.* Suppose $$K\triangleleft G$$ is a normal subgroup in cyclic group $$G$$, and $$x$$ is the generator. Then for any element $$aK$$ in the quotient group, $$aK=(x^n)K=(xK)^n$$ for some integer $$n$$. Thus the quotient group is cyclic.

**Definition** (Simple group)**.** A group is *simple* if it has no non-trivial proper normal subgroup, i.e. only $$\{e\}$$ and $$G$$ are normal subgroups.    
**Proposition.** A group $$G$$ of order 6 is either cyclic or dihedral, $$G\cong C_6$$ or $$D_6$$.

> *Proof.* Let $$|G|=6$$. By Lagrange's theorem, possible element orders are $$1,2,3,6$$. If $$a\in G$$ is of order $$6$$, then $$G=\langle a\rangle\cong C_6$$.     
> Suppose there's only elements of order $$2$$ and $$3$$. If $$G$$ only has elements of order $$2$$, the order of the group must be a power of 2, which is not the case. So there must be an element $$r$$ of order $$3$$. Thus $$\langel r\rangle\triangleleft G$$ as it has index $$2$$. Deote $$s$$ is an element of order $$2$$. Then $$srs^{-1}\in\langle r\rangle$$. If $$srs^{-1}=e$$, then $$r=e$$, not true. If $$srs^{-1}=r$$, then $$sr=rs$$ and $$sr$$ has order $$6$$ (lcm of the orders of $$s$$ and $$r$$), not possible. Thus $$srs^{-1}=r^2=r^{-1}$$, and $$G\cong D_6$$.

#### *Group Acting on Sets, Orbit and Stabiliser*
**Definition** (Group action)**.** Let $$X$$ be a set and $$G$$ be a group. An *action* of $$G$$ on $$X$$ is a homomorphism $$\phi:G\to\text{Sym}X$$.    
The homomorphism $$\phi$$ turns each element $$g\in G$$ into a permutation of $$X$$. This could be written as $$\phi(g)(x),g(x)$$ or $$gx$$.        
**Proposition.** Let $$X$$ be a set and $$G$$ be a group. Then $$\phi:G\to\text{Sym}X$$ is a homomorphism (an action) iff $$\theta:G\times X\to X$$ defined by $$\theta(g,x)=\phi(g)(x)$$ satisfies     
&emsp; 1. $$\forall g\in G, x\in X,\theta(g,x)\in X.$$    
&emsp; 2. $$\forall x\in X,\theta(e,x)=x.$$    
&emsp; 3. $$\forall g,h\in G,\forall x\in X,\theta(g,\theta(h,x))=\theta(gh,x).$$    
**Defintion** (Kernel of action)**.** The *kernel* of an action $$G$$ on $$X$$ is the kernel of $$\phi$$, i.e. all $$g$$ such that $$\phi(g)=1_X$$.    
Note that $$\text{ker}\phi\triangleleft G$$ and $$G/K$$ is isomorphism to a subgroup of $$\text{Sym}X$$.    
**Definition** (Faithful action)**.** An action is *faithful* if the kernel is just $$\{e\}$$.    
**Definition** (Orbit of action)**.** Given an action $$G$$ on $$X$$, the *orbit* of an element $$x\in X$$ is

$$\text{Orb}(x)=G(x)=\{y\in X:\exists g\in G,g(x)=y\}.$$

**Definition** (Stabiliser of action)**.** The *stabiliser* of $$x$$ is

$$\text{Stab}(x)=G_x=\{g\in G:g(x)=x\}\subseteq G.$$

**Lemma.** $$\text{Stab}(x)$$ is a subgroup of $$G$$.    

### <center>Lecture 12. Orbits and Some Important Actions</center>
**Definition** (Tansitive action)**.** An action $$G$$ on $$X$$ is *transitive* if $$\forall x\in X,\text{Orb}(x)=X.$$    
**Lemma.** The orbits of an action partition $$X$$.    
**Theorem** (Orbit-stabiliser theorem)**.** Let the group $$G$$ act on $$X$$. There is a bijection between $$\text{Orb}(x)$$ and cosets of $$\text{Stab}(x)$$ in $$G$$. In particular, if $$G$$ is finite, then

$$|\text{Orb}(x)||\text{Stab}(x)|=|G|.$$

**Example.** 1. $$D_{2n}$$ acts on the vertices $$\{1,2,\cdots,n\}$$. $$|\text{Orb}(1)|=n$$. And $$\text{Stab}(1)=\{e,\text{reflection through 1}\}$$. Thus $$|D_{2n}|=2n$$.    
&emsp;2. Let $$\langle(12)\rangle$$ act on $$\{1,2,3\}$$. Then $$\text{Orb}(1)=\{1,2\}$$, $$\text{Stab}(1)=\{e\}$$. $$\text{Orb}(3)=\{3\}$$, $$\text{Stab}(3)=\langle(12)\rangle$$.     
&emsp;3. Let $$S_4$$ act on $$\{1,2,3,4\}$$. Know that $$\text{Orb}(1)=X$$ and $$|S_4|=24$$. So $$|\text{Stab}(1)|=6$$. Clearly $$S_\{2,3,4\}\cong S_3$$ fix 1. Thus $$S_3\leq \text{Stab}(1)$$. However $$|S_3|=|\text{Stab}(1)|=6$$. So this is all of the stabiliser.

### <center>Lecture 13. Conjugacy Classes, Centralisers and Normalisers</center>
**Definition** (Conjugacy classes and centralisers)**.** The *conjugacy classes* are the orbits of the conjugacy action.

$$\text{ccl}(a)=\{b\in G:\exists g\in G, gag^{-1}=b\}.$$

The *centralisers* are the stabilisers of this action, i.e. elements that commute with $$a$$.

$$C_G(a)=\{g\in G:gag^{-1}=a\}=\{g\in G:ga=ag\}.$$

The centraliser is defined as the elements that commute with a particular element $$a$$.    
**Lemma.** Let $$K\triangleleft G$$. Then $$G$$ acts by conjugation on $$K$$.       
**Proportion.** Normal subgroups are exactly those subgroups which are unions of conjugacy classes.    
**Lemma.** Let $$X$$ be the set of subgroups of $$G$$. Then $$G$$ acts by conjugation on $$X$$.    
**Definition** (Normaliser of subgroup)**.** The *normaliser* of a subgroup is the stabiliser of the group conjugation action.

$$N_G(H)=\{g\in G:gHg^{-1}=H\}.$$

Clearly $$H\subseteq N_G(H)$$. $$N_G(H)$$ is the largest subgroup of $$G$$ in which $$H$$ is a normal subgroup.    
There is a connection between actions in general and conjugation of subgroups.     
**Lemma.** Stabilisers of the elements in the same orbit are conjugate, i.e. let $$G$$ act on $$X$$ and let $$g\in G,x\in X$$. Then $$\text{Stab}(g(x))=g\text{Stab}(x)g^{-1}.$$

### <center>Lecture 15. Rings, Fields and Polynomials</center>
**Definition** (Ring)**.** A *ring* is a quintuple $$(R,+,\cdot,0_R,1_R)$$ where $$0_R,1_R\in R$$, and $$+,\cdot:R\times R\to R$$ are binary operations such that     
&emsp; 1. $$(R,+,0_R)$$ is an abelian group;    
&emsp; 2. The operation $$\cdot:R\times R\to R$$ satisfies associativity, i.e.

$$a\cdot (b\cdot c)=(a\cdot b)\cdot c,$$

&emsp; and the identity:

$$1_R\cdot r=r\cdot 1_R=r.$$

&emsp; 3. Multiplication distributes over addition, i.e.

$$\begin{align*}
&r_1\cdot(r_2+r_3)=(r_1\cdot r_2)+(r_1\cdot r_3),\\
&(r_1+r_2)\cdot r_3 =(r_1\cdot r_3)+(r_2\cdot r_3).
\end{align*}$$

**Notation.** If $$R$$ is a ring and $$r\in R$$, we write $$-r$$ for inverse to $$r$$ in $$(R,+,0_R)$$. Write $$r-s$$ to mean $$r+(-s)$$.    
**Definition** (Commutative ring)**.** Ring $$R$$ is *commutative* if $$a\cdot b=b\cdot a$$ for all $$a,b\in R$$.    
From now on all rings are commutative.    
**Definition** (Subring)**.** Let $$(R,+,\cdot,0_R,1_R)$$ be a ring and $$S\subseteq R$$ be a subset. $$S$$ is a *subring* of $$R$$ if $$0_R,1_R\in S$$ and the operations $$+,\cdot $$ make $$S$$ into a ring in its own right. Write as $$S\leq R$$.    
**Example.** $$\mathbb{Z}\leq \mathbb{Q}\leq \mathbb{R}\leq \mathbb{C}$$ under $$0,1,+,\cdot$$.    
**Example.** The set $$\mathbb{Z}[i]=\{a+ib:a,b\in\mathbb{Z}\}\leq \mathbb{C}$$ (Gaussian integers) is ring. And $$\mathbb{Q}[\sqrt{2}]=\{a+b\sqrt{2}\in\mathbb{R}:a,b\in\mathbb{Q}\}\leq \mathbb{R}$$ is a ring.      
Generally elements in a ring do not have inverses.    
**Definition** (Unit)**.** An element $$u\in R$$ is a *unit* if there is another element $$v\in R$$ such that $$u\cdot v=1_R.$$     
**Definition** (Field)**.** A *field* is a non-zero ring where all $$u\neq 0_R\in R$$ is a unit.     
**Example.** $$\mathbf{Q},\mathbf{R},\mathbf{C}$$ are fields, $$\mathbf{Z}$$ is not. $$\mathbf{Q}[\sqrt{2}]$$ is a field, $$\mathbf{Z}[i]$$ is not.     
**Definition** (Product of rings)**.** Let $$R,S$$ be rings. Then the product $$R\times S$$ is a ring via 

$$(r,s)+(r',s')=(r+r',s+s'),\quad (r,s)\cdot(r',s')=(r\cdot r',s\cdot s').$$

The zero is $$(0_R,0_S)$$ and the one $$(1_R,1_S).$$     
**Definition** (Polynomial)**.** Let $$R$$ be a ring. Then a *polynomial* with coefficients in $$R$$ is an expression 

$$f=a_0+a_1X+a_2X^2+\cdots+a_nX^n,$$

with $$a_i\in R.$$     
**Definition** (Degree of polynomial)**.** The *degree* of a polynomial $$f$$ is the largest $$m$$ such that $$a_m\neq 0$$.     
**Definition** (Monic polynomial)**.** $$f$$ has degree $$m$$. If $$a_m=1$$, then $$f$$ is called *monic*.      
**Definition** (Polynomial ring)**.** Write $$R[X]$$ for the set of all polynomials with coefficients in $$R$$. If $$f=a_0+a_1X+\cdots+A_nX^n$$ and $$g=b_0+b_1X+\cdots+b_kX^k$$ are polynomials. Then

$$f+g=\sum_{r=0}^{\max\{n,k\}}(a_i+b_i)X^i,\quad f\cdot g=\sum_{i=0}^{n+k}\sum_{j=0}^i a_jb_{i-j}X^i.$$

$$0_R,1_R\in R$$ are the zero and one of the $$R[X]$$. $$R[X]$$ is a ring.    
**Definition** (Power series)**.** Write $$R[[X]]$$ for the ring of power series on $$R$$, i.e.

$$f=a_0+a_1X+a_2X^2+\cdots,$$

where each $$a_i\in R$$. This has addition and multiplication the same for polynomials, but without upper limits.    
**Example.** In $$R[X]$$, $$1-X$$ is not a unit. For every $$g=a_0+\cdots+a_nX^n$$ (with $$a_n\neq0$$), we have

$$(1-X)g=\cdots-a_nX^{n+1}.$$

However in $$R[[X]]$$, $$1-X$$ is a unit since

$$(1-X)(1+X+X^2+X^3+\cdots)=1.$$

### <center>Lecture 16 Homomorphisms, Ideals, Quotients and Isomorphisms</center>
**Definition** (Homomorphism of rings)**.** Let $$R,S$$ be rings. A function $$\phi:R\to S$$ is a *ring homomorphism* if     
&emsp; 1. $$\phi(r_1+r_2)=\phi(r_1)+\phi(r_2),$$     
&emsp; 2. $$\phi(0_R)=0_S,$$      
&emsp; 3. $$\phi(r_1\cdot r_2)=\phi(r_1)\cdot\phi(r_2),$$     
&emsp; 4. $$\phi(1_R)=1_S.$$     
**Definition** (Isomorphism of rings)**.** If a homomorphism $$\phi:R\to S$$ is a bijection, we call it an *isomorphism*.      
**Definition** (Kernel)**.** The *kernel* of a homomorphism $$\phi:R\to S$$ is

$$\ker(\phi)=\{r\in R:\phi(r)=0_S\}.$$

**Definition** (Image)**.** The *image* of $$\phi:R\to S$$ is

$$\text{Im}(\phi)=\{s\in S:\exists r\in R, s=\phi(r)\}.$$

**Lemma.** A homomorphism $$\phi:R\to S$$ is injective iff $$\ker\phi=\{0_R\}.$$   
**Definition** (Ideal)**.** A subset $$I\subseteq R$$ is an *ideal* written $$I\triangleleft R$$ if     
&emsp; 1. (Additive closure) It is an additive subgroup of $$(R,+,0_R)$$, i.e. it's closed under addition and additive inverses.     
&emsp; 2. (Strong closure) If $$a\in I$$ and $$b\in R$$, then $$a\cdot b\in I.$$        
&emsp; $$I$$ is a *proper* ideal if $$I\neq R.$$       
**Lemma.** If $$\phi:R\to S$$ is a homomorphism, then $$\ker(\phi)\triangleleft R$$.     
> *Proof.* Since $$\phi:(R,+,0_R)\to (S,+,0_S)$$ is a group homomorphism, then the kernel is a subgroup of $$(R,+,0_R)$$.       
> Let $$a\in\ker(\phi),b\in R$$. And then $$\phi(a\cdot b)=\phi(a)\cdot\phi(b)=0\cdot\phi(b)=0.$$ Thus $$a,b\in\ker(\phi).$$

**Example.** Suppose $$I\triangleleft R$$ and $$1_R\in I$$. Then $$\forall r\in R, 1_R\cdot r=r\in I\Rightarrow I=R$$. That is, every proper ideal doesn't contain $$1$$, and every proper ideal is not a subring.      
**Example.** Suppose $$I\triangleleft R$$ and $$u\in I$$ is a unit, i.e. there is some $$v\in R$$ such that $$u\cdot v=1_R$$. They by strong closure, $$1_R=u\cdot v\in I$$, then $$I=R$$. Hence proper ideals are not allowed to contain any unit at all.      
**Example.** Consider the ring $$\mathbb{Z}$$. Every ideal of $$\mathbb{Z}$$ is of the form

$$n\mathbb{Z}=\{\cdots,-2n,-n,0,n,2n,\cdots\}\subseteq\mathbb{Z}.$$

**Definition** (Generator of ideal)**.** For an element $$a\in R$$, we write

$$(a)=aR=\{a\cdot r:r\in R\}\triangleleft R.$$

This is the *ideal generated by* $$a$$. In general, let $$a_1,a_2,\cdots, a_k\in R$$, we write

$$(a_1,a_2,\cdots, a_k)=\{a_1r_2+\cdots+a_kr_k:r_1,\cdots,r_k\in R\}.$$

This is the *ideal generated by* $$a_,\cdots,a_k$$.      
**Definition** (Generator of ideal)**.** For $$A\subseteq R$$ a subset, the *ideal generated by* $$A$$ is

$$(A)=\left\{\sum_{a\in A}r_a\cdot a:r_a\in R,\text{only finitely-many non-zero}\right\}.$$

**Definition** (Principle ideal)**.** An ideal $$I$$ is a *principle ideal* if $$I=(a)$$ for some $$a\in R$$.     
**Definition** (Quotient group)**.** Let $$I\triangleleft R$$. The *quotiont ring* $$R/I$$ consists of the additive costs $$r+I$$ with the zero and one as $$0_R+I$$ and $$1_R+I$$ and operations

$$\begin{align*}
(r_1+I)+(r_2+I)&=(r_1+r_2)+I,\\
(r_1+I)\cdot (r_2+I)&=r_1r_2+I.
\end{align*}$$

**Proposition.** The quotient ring is a ring, and the function $$R\to R/I, r\mapsto r+I$$ is a ring homomorphism.       
> *Proof.* The group $$(R/I,+,0_{R/I})$$ is well-defined since $$I$$ is a normal subgroup of $$R$$. Now we check multiplication is well-defined. Suppose $$r_1+I=r_1'+I$$ and $$r_2+I=r_2'+I$$. Then $$r_1'-r_1=a_1\in I$$ and $$r_2-r_2'=a_2\in I$$. Thus
> 
> $$r_1'r_2'=r_1r_2+r_1a_2+r_2a_1+a_1a_2.$$
> 
> Since the last three terms is in $$I$$, thus $$r_1'r_2'+I=r_1r_2+I.$$ Hence the multiplication is well-defined. 

**Example.** The ideals $$n\mathbb{Z}\triangleleft\mathbb{Z}$$. The quotient rings $$\mathbb{Z}/n\mathbb{Z}$$. The elements are of the form $$m+n\mathbb{Z}$$:

$$0+n\mathbb{Z},1+n\mathbb{Z},2+n\mathbb{Z},\cdots, (n-1)+n\mathbb{Z}.$$

Addition and multiplication are the addition and multiplication modulo $$n$$.    

### <center>Lecture 17. Euclidean Algorithm and Three Isomoprhism Theorems</center>
**Proposition** (Euclidean alorithm for polynomials)**.** Let $$\mathscr{F}$$ be a field and $$f,g\in\mathscr{F}[X]$$. Then there is some $$r,q\in \mathscr{F}[X]$$ such that

$$f=gq+r,$$

with $$\text{deg }r<\text{deg }g.$$     
**Example.** Consider $$\mathbb{R}[X]$$ and consider the principal ideal $$(X^2+1)\triangleleft\mathbb{R}[X]$$. Let $$R=\mathbb{R}[X]/(X^2+1).$$ Elements of $$R$$ are the polynomials

$$\underbrace{a_0+a_1X+a_2X^2+\cdots+a_nX^n}_f+(X^2+1).$$

By the Euclidean algorithm, we have

$$f=q(X^2+1)+r,$$

with $$\text{deg}(r)<2, r=b_0+b_1X.$$ Thus $$f+(X^2+1)=r+(X^2+1).$$ So every element of $$\mathbb{R}[X]/(X^2+1)$$ is representable as $$a+bX$$ for some $$a,b\in\mathbb{R}$$. The representation is unique. Suppose $$a+bX+(X^2+1)=a'+b'X+(X^2+1)$$, then $$(a-a')+(b-b')X\in (X^2+1)\Rightarrow(a-a')+(b-b')X=(X^2+1)q$$. $$q$$ has to be $$0$$ since $$(X^2+1)q$$ has degree at least $$2$$. Thus $$a+bX=a'+b'X$$ and theh representation is unique. Now every element in $$R$$ is of the form $$a+bX$$ and $$X^2+1=0$$. Consider the function $$\phi: \mathbb{R}[X]/(X^2+1)\to\mathbb{C}$$ with $$a+bX+(X^2+1)\mapsto a+bi.$$ This is well-defined and a bijection. 

**Theorem** (First isomorphism theorem)**.** Let $$\phi:R\to S$$ be a ring homomorphism. Then $$\ker(\phi)\triangleleft R$$ and 

$$\frac{R}{\ker(\phi)}\cong\text{Im}(\phi)\leq S.$$

> *Proof.* We've already proved that $$\ker(\phi)\triangleleft R$$. Now define $$\Phi:R/\ker(\phi)\to\text{Im}(\phi)$$ with $$r+\ker(\phi)\mapsto\phi(r)$$. This is well-defined since if $$r+\ker(\phi)=r'+\ker(\phi)$$, then $$r-r'\in\ker(\phi)$$. Thus $$\phi(r-r')=0,\phi(r)=\phi(r')$$. $$\Phi$$ is bijective and additive (comes from the proof of the isomorphism theorem of groups.) So only need to check it's multiplicative:
> 
> $$\begin{align*}\Phi((k+\ker(\phi))(t+\ker(\phi))))&=\Phi(rt+\ker(\phi))\\&=\phi(rt)=\phi(r)\phi(t)\\&=\Phi(r+\ker(\phi))\Phi(t+\ker(\phi))\end{align*}$$
>

**Theorem** (Second isomorphism theorem)**.** Let $$R\leq S$$ and $$J\triangleleft S$$. Then $$J\cap R\triangleleft R$$ and 

$$\frac{R+J}{J}=\{r+J:r\in R\}\leq\frac{S}{J}$$

is a subring, and 

$$\frac{R+J}{J}=\{r+J:r\in R\}\leq\frac{S}{J}.$$

> *Proof.* Define the function $$\phi:R\to S/J$$ with $$r\mapsto r+J$$. Since this is a quotient map, it's a ring homomorphism. Then the kernel is $$\ker(\phi)=\{r\in R:r+J=J\Leftrightarrow r\in J\}=R\cap J$$. And the image is $$\text{Im}(\phi)=\{r+J:r\in R\}=(R+J)/J.$$ By the first isomorphism theorem, $$R\cap J\triangleleft R$$ and $$(R+J)/J\leq S$$ and $$R/(R\cap J)\cong (R+J)/J$$.

**Theorem** (Third isomorphism theorem)**.** Let $$I\triangleleft R, J\triangleleft R, I\subseteq J$$. Then $$J/I\triangleleft R/I$$ and 

$$\left(\frac{R}{I}\right)\big/\left(\frac{J}{I}\right)\cong\frac{R}{J}.$$

> *Proof.* Define the map $$\phi:R/I\to R/J$$ with $$r+I\mapsto r+J$$. It's well-defined and surjective by the group case. It's also a ring homomorphism since multiplication in $$R/I$$ and $$R/J$$ are "the same". The kernel is $$\ker(\phi)=\{r+I:r+J=0\Leftrightarrow r\in J\}=J/I.$$

### <center>Lecture 18. Integral Domains, Field of Fractions</center>
**Definition** (Integral domain)**.** A non-zero commutative ring $$R$$ is an *integral domain* if for all $$a,b\in R$$, $$a\cdot b=0_R\Rightarrow a=0_R$$ or $$b=0_R$$.      

> Suppose $$R$$ is an integral domain. If $$a,b,c\in R, a\neq 0$$, then $$ab=ac\Rightarrow b=c$$.     
> *Proof.* $$ab=ac\Rightarrow ab-ac=0\Rightarrow a(b-c)=0.$$ By the definition of integral domain, we have $$b-c=0\Rightarrow b=c$$. 

**Definition** (Zero divisor)**.** An element $$x\in R$$ is a *zero divisor* if $$x\neq 0$$ and there is a $$y\neq 0$$ such that $$x\cdot y=0\in R$$.     
A ring is an integral domain if it has no zero divisor.    
**Example.** All fields are integral domain, since if $$a\cdot b=0$$, and $$b\neq 0$$, then $$a=a\cdot(b\cdot b^{-1})=0$$. Similarly, if $$a\neq 0$$, then $$b=0$$.     
**Example.** A subring of an integral domain is an integral domain, since a zero divisor in the small ring is also be a zero divisor in the big ring. (In the big ring there's no zero divisor, then in the small ring there is also no zero divisor.)       
**Example.** $$\mathbb{Z},\mathbb{Q},\mathbb{R},\mathbb{C}$$ are integral domains, since $$\mathbb{C}$$ is a field and the others are subrings of it. Also $$\mathbb{Z}[i]\leq \mathbb{C}$$ is also an integral domain.      
**Lemma.** Let $$R$$ be a finite ring which is an integral domain. Then $$R$$ is a field.    

> *Proof.* Let $$a\in R$$ be a non-zero element, and consider the ring homomorphism $$a\cdot-:R\to R$$ with $$b\mapsto a\cdot b$$. If $$r\in\ker(a\cdot-)$$, then $$a\cdot r=0$$. Thus $$r=0$$ since $$R$$ is an integral domain. So the kernel is trivial and thus $$a\cdot-$$ is injective. Since $$R$$ is finite, $$a\cdot -$$ must be bijective. Particularly there is a $$b\in R$$ such that $$ab=1$$. Thus $$a$$ has an inverse. Since $$a$$ is arbitrary, $$R$$ is a field.

**Lemma.** Let $$R$$ be an integral domain. Then $$R[X]$$ is also an integral domain.    

> *Proof.* Need to show that the product of two non-zero elements is non-zero. Let $$f,g\in R[X]$$ be non-zero, say $$f=a0+a_1X+\cdots+a_nX^n,g=b_0+b_1X+\cdots+b_mX^m$$ with $$a_n,b_m\neq 0$$. Then the coefficient of $$X^{m+n}$$ in $$fg$$ is $$a_nb_m$$. This is non-zero since $$R$$ is an integral domain. Thus $$fg$$ is non-zero. And $$R[X]$$ is an integral domain.

Write $$R[X,Y]$$ for the polynomial ring of $$R$$ in 2 variables. In general, if $$R$$ is an integral domain, then so is $$R[X_1,\cdots,X_n]$$.     
**Definition** (Field of fractions)**.** Let $$R$$ be an integral domain. A *field of fractions* $$F$$ of $$R$$ is a field with the following properties     
&emsp; 1. $$R\leq F$$;    
&emsp; 2. Every element of $$F$$ may be written as $$a\cdot b^{-1}$$ for $$a,b\in R$$, where $$b^{-1}$$ means the multiplicative inverse to $$b\neq 0$$ in $$F$$.      
$$\mathbb{Q}$$ is the field of fractions of $$\mathbb{Z}$$.     
**Theorem.** Every integral domain has a field of fractions.    

> *Proof.* Let $$S=\{(a,b)\in R\times R:b\neq 0\}$$. Define the equivalence relation $$\sim$$ on $$S$$ by $$(a,b)\sim(c,d)\Leftrightarrow ad=bc$$. Need to show this is an equivalence relation. Symmetry and reflexivity are obvious. To show transitivity, suppose $$(a,b)\sim(c,d),(c,d)\sim(e,f)$$, i.e. $$ad=bc,cf=de\Rightarrow adf=bcf,bcf=bed$$. Thus $$d(af-be)=0$$. Note that $$d\neq 0$$, and $$R$$ is an integral domain, thus $$af-be=0\Rightarrow af=be$$.     
> Now let $$F=S/\sim$$ be the set of equivalence classes. Need to check this is the field of fractions. Write $$\frac{a}{b}=[(a,b)]\in F$$. Define the operations by $$\frac{a}{b}+\frac{c}{d}=\frac{ad+bc}{bd},\quad \frac{a}{b}\cdot \frac{c}{d}=\frac{ac}{bd}.$$ These are well-defined and make $$(F,+,\cdot,\frac{0}{1},\frac{1}{1})$$ into a ring.     
> Need to show every non-zero element has an inverse. Let $$\frac{a}{b}\neq 0_F=\frac{0}{1},$$ or $$a\cdot 1\neq b\cdot 0\in R\rightarrow a\neq 0$$. Then $$\frac{b}{a}\in F$$ is defined, and $$\frac{b}{a}\cdot\frac{a}{b}=1_F.$$ Thus $$F$$ is a field.    
> Construct a subring of $$F$$ which is isomorphic to $$R$$. Define a injective isomorphism $$\phi:R\to F$$. Given $$\phi:R\to F, r\mapsto\frac{r}{1}$$. This is a ring homomorphism. The kernel is trivial, and $$\phi$$ is injective. By the first isomorphism theorem, $$R\cong\text{Im}(\phi)\subseteq F$$.    
> Finally need to show everything is a quotient of two things in $$R$$. We have $$\frac{a}{b}=\frac{a}{1}\cdot\frac{1}{b}=\frac{a}{1}\left(\frac{b}{1}\right)^{-1}$$ as required.

**Example.** Consider the integral domain $$\mathbb{C}[X]$$. Its field of fractions is the field of all rational functions $$\frac{p(X)}{q(X)}$$ where $$p,q\in \mathbb{C}[X]$$.

### <center>Lecture 19. Maximal and Prime Ideals</center>
**Lemma.** A (non-zero) ring is a field iff its only ideals are $$\{0\}$$ and $$R$$.    

> *Proof.* $$(\Rightarrow)$$ Let $$I\triangleleft R$$ and $$R$$ is a field. Suppose $$x\neq 0\in I$$. Then since $$x$$ is a unit, $$I=R$$.     
> $$(\Leftarrow)$$ Suppose $$x\neq 0\in R$$. Then $$(x)$$ is an ideal of $$R$$, it's not $$\{0\}$$ since it contains $$x$$. Then $$(x)=R$$. This means that $$1_R\in(x)$$. However, $$(x)=\{x,y:y\in R\}$$. Thus there is some $$u\in R$$ such that $$x\cdot u=1_R$$, i.e. $$x$$ is a unit. Since $$x$$ is arbitrary, $$R$$ is a field.

**Definition** (Maximal ideal)**.** An ideal $$I$$ of a ring $$R$$ is *maximal* if $$I\neq R$$ for any ideal $$J$$ with $$I\leq J\leq R$$, either $$J=I$$ or $$J=R$$.    

> In other words, $$I$$ is a maximal ideal of a ring $$R$$ if there are no other ideals contained between $$I$$ and $$R$$.

**Lemma.** An ideal $$I\triangleleft R$$ is maximal iff $$R/I$$ is a field.    

> *Proof.* $$R/I$$ is a field iff $$\{0\},R/I$$ are the only ideals of $$R/I$$. By the ideal correspondence, this is equivalent to saying $$I$$ and $$R$$ are the only ideals of $$R$$ which contains $$I$$, i.e. $$I$$ is maximal.

> **Ring Correspondence Theorem**. we can construct a bijection from the set of all subrings of $$R$$ which contain an ideal $$I$$, and the subrings of $$R/I$$. In particular, the bijection is the map $$J\mapsto J/I$$, where $$I\subseteq J$$ and $$J$$ is a subring of $$R$$ which contains $$I$$.

**Definition** (Prime ideal)**.** An ideal $$I$$ of a ring $$R$$ is *prime* if $$I\neq R$$ and whenever $$a,b\in R$$ are such that $$a\cdot b\in I$$, then $$a\in I$$ or $$b\in I$$.    
**Example.** A non-zero ideal $$n\mathbb{Z}\triangleleft\mathbb{Z}$$ is prime iff $$n$$ is prime.    
**Lemma.** An ideal $$I\triangleleft R$$ is prime iff $$R/I$$ is an integral domain.    

> *Proof.* Let $$I$$ be prime. Let $$a+I,b+I\in R/I$$. Suppose $$(a+I)(b+I)=ab+I=0$$. Thus $$ab\in I$$. Then either $$a\in I$$ or $$b\in I$$. This means that $$a+I=0$$ or $$b+I=0$$. Thus $$R/I$$ is an integral domain.     
> Conversely, suppose $$R/I$$ is an integral domain. Let $$a,b\in R$$ and $$ab\in I$$. Then $$(a+I)(b+I)=ab+I=0$$. Since $$R/I$$ is an integral domain, either $$a+I=0$$ or $$b+I=0$$, i.e. $$a\in I$$ or $$b\in I$$. Thus $$I$$ is prime. 

**Proposition.** Every maximal ideal is a prime ideal. (The converse is not true).  

> *Proof.* $$I\triangleleft R$$ is maximal $$\Rightarrow R/I$$ is a field $$\Rightarrow R/I$$ is an integral domain $$\Rightarrow I$$ is prime. 

For any ring $$R$$, there is a unique ring homomorphism $$\mathbb{Z}\to R$$ given by

$$\begin{align*}
\imath:\mathbb{Z}&\to R\\
n\geq 0&\mapsto \underbrace{1_R+1_R+\cdots+1_R}_{n\text{ times }}\\
n\leq 0&\mapsto -\underbrace{1_R+1_R+\cdots+1_R}_{-n\text{ times }}\\
\end{align*}$$

Note that $$1_R+1_R+\cdots+1_R=0_R$$ for $$n$$ $$1_R$$. Thus $$\ker(\imath)=n\mathbb{Z}$$ for some $$n$$.    
**Definition** (Characteristic of ring)**.** Let $$R$$ be a ring and $$\imath:\mathbb{Z}\to R$$ be the unique such map. The *characteristic* of $$R$$ is the unique non-negative $$n$$ such that $$\ker(\imath)=n\mathbb{Z}.$$       
**Example.** The rings $$\mathbb{Z,Q,R,C}$$ all have characteristic $$0$$. The ring $$\mathbb{Z}/n\mathbb{Z}$$ has charactersitic $$n$$. In particular, all natural numbers can be characteristics.    
**Lemma.** Let $$R$$ be an integral domain. Then its characteristic is either $$0$$ or a prime number.    

> *Proof.* Consider the unique map $$\phi:\mathbb{Z}\to R$$ and $$\ker(\phi)=n\mathbb{Z}$$. Then $$n$$ is the characteristic of $$R$$. By the frist isomorphism theorem, $$\mathbb{Z}/n\mathbb{Z}=\text{Im}(\phi)\leq R.$$ Thus $$\mathbb{Z}/n\mathbb{Z}$$ is an integral domain. Thus $$n\mathbb{Z}$$ is prime, and hence $$n$$ is either $$0$$ or a prime number.

### <center>Lecture 20. Factorisation in Integral Domains</center>
Suppose $$R$$ is an integral domain.    
**Definition** (Unit)**.** An element $$a\in R$$ is a *unit* if there is a $$b\in R$$ such that $$ab=1_R$$. Equivalently, if the ideal $$(a)=R$$.       
**Definition** (Division)**.** For elements $$a,b\in R$$, we say $$a$$ *divides* $$b$$, written $$a|b$$ if there is a $$c\in R$$ such that $$b=ac$$. Equivalently, if $$(b)\subseteq (a)$$.     
**Definition** (Associates)**.** We say $$a,b\in R$$ are *associates* if $$a=bc$$ for some unit $$c$$. Equivalently, if $$a|b$$ and $$b|a$$.    
**Definition** (Irreducible)**.** $$a\in R$$ is *irreducible* if $$a\neq 0$$, $$a$$ is not a unit, and if $$a=xy$$, then $$x$$ or $$y$$ is a unit.    
**Definition** (Prime)**.** $$a\in R$$ is *prime* if $$a$$ is non-zero, not a unit, and whenever $$a|xy$$, either $$a|x$$ or $$a|y$$.    
**Lemma.** A principal ideal $$(r)$$ is a prime ideal in $$R$$ iff $$r=0$$ or $$r$$ is prime.       

> *Proof.* $$(\Rightarrow)$$ Let $$(r)$$ be a prime ideal. If $$r=0$$ then we are done. Otherwise, as prime ideals are proper (not the whole ring), $$r$$ is not a unit. Now suppose $$r|ab$$. Then $$ab\in (r)$$. But $$(r)$$ is prime. So $$a\in(r)$$ or $$b\in(r)$$. Thus $$r|a$$ or $$r|b$$. So $$r$$ is prime.     
> $$(\Leftarrow)$$ If $$r=0$$, then $$(0)=\{0\}\triangleleft R$$ is prime since $$R/\{0\}=R$$ is an integral domain. Otherwise, let $$r\neq 0$$ be prime. Suppose $$ab\in (r)$$. This means that $$r|ab$$. Thus $$r|a$$ or $$r|b$$. So $$a\in(r)$$ or $$b\in(r)$$. Thus $$(r)$$ is prime.

**Lemma.** Let $$r\in R$$ be prime. Then it's irreducible.     

> *Proof.* Suppose $$r=ab\in R$$. Since $$r|ab$$ and $$r$$ is prime, we have $$r|a$$ or $$r|b$$. Assume $$r|a$$. Then $$a=rc$$ for some $$c\in R$$. Thus $$r=ab=rcb$$. Thus $$cb=1$$. So $$b$$ is a unit. Similar for $$r|b$$. 

**Example.** Let $$R=\mathbb{Z}[\sqrt{-5}]=\{a+b\sqrt{-5}:a,b\in\mathbb{Z}\}\leq \mathbb{C}.$$ It's a subring of a field, and thus an integral domain. To find the units of the ring, consider the norm $$N:R\to\mathbb{Z}_{\geq 0}$$ given by $$N(a+b\sqrt{-5})\mapsto a^2+5b^2. This satisfies $$N(zw)=N(z)N(w)$$. All units have norm $$1$$. 
**Definition** (Euclidean domain)**.** An integral domain $$R$$ is a *Euclidean domain* (ED) if there is a *Euclidean function* $$\phi:R\backslash\{0\}\to\mathbb{Z}_{\geq 0}$$ such that     
&emsp; 1. $$\phi(a\cdot b)\geq \phi(b)$$ for all $$a,b\neq 0$$;      
&emsp; 2. If $$a,b\in R$$ with $$b\neq 0$$, then there are $$q,r\in R$$ such that $$a=b\cdot q+r$$, and either $$r=0$$ or $$\phi(r)<\phi(b)$$.     
**Example.** $$\mathbb{Z}$$ is a Euclidean domain with $$\phi(n)=|n|$$.    
**Example.** For any field $$\mathscr{F}$$, $$\mathscr{F}[X]$$ is a Euclidean domain with $$\phi(f)=\text{deg}(f)$$.     
**Example.** The Gaussian integers $$R=\mathbb{Z}[i]\leq\mathbb{C}$$ is a Euclidean domain with $$\phi(z)=N(z)=|z|^2$$.    


### <center>Lecture 21. Principal Ideal Domain and Unique Factorisation Domain</center>
**Definition** (Principal ideal domain)**.** A ring $$R$$ is a *principal ideal domain* (PID) if it is an integral domain, and every ideal is a principal ideal, i.e. for all $$I\triangleleft R$$, there is some $$a$$ such that $$I=(a)$$.    
**Example.** $$\mathbb{Z}$$ is a principal ideal domain.     
**Proposition.** Let $$R$$ be a Euclidean domain. Then $$R$$ is a principal ideal domain.    

> *Proof.* Let $$R$$ have a Euclidean function $$\phi:R\\{0\}\to\mathbb{Z}_{\geq 0}$$. Let $$I\triangleleft R$$ be a non-zero ideal, and let $$b\in I\\{0\}$$ be an element with $$\phi(b)$$ minimal. Then for any $$a\in I$$, $$a=bq+r$$, with $$r=0$$ or $$\phi(r)<\phi(b)$$. Note that $$r=a-bq\in I$$. Thus we cannot have $$\phi(r)<\phi(b)$$, and thus $$r=0$$. So $$a=bq\Rightarrow a\in(b)$$. Since this is true for all $$a\in I$$, $$I\subseteq (b)$$. Also since $$b\in I$$, we must have $$(b)\subseteq I$$. Thus $$I=(b)$$. This works for any ideal in $$R$$ and thus $$R$$ is a principal domain.

**Example.** $$\mathbb{Z},\mathscr{F}[X],\mathbb{Z}[i]$$ are Euclidean domains, and thus PIDs.     
**Definition** (Unique factorisation domain)**.** An integral domain $$R$$ is a *unique factorisation domain* (UFD) if      
&emsp; 1. Every non-unit may be written as product of irreducibles;    
&emsp; 2. If $$p_1p_2\cdots p_n=q_1\cdots q_m$$ with $$p_i,q_j$$ irreducibles, then $$n=m$$ and they can be reordered suchh that $$p_i$$ is an associate of $$q_i$$.      
**Lemma.** Let $$R$$ be a principal ideal domain. If $$p\in R$$ is irreducible, then it is prime.     

> *Proof.* Let $$p\in R$$ be irreducible, and suppose $$p|ab$$. Suppose $$p\nmid a$$, we need to show $$p|b$$. Consider the ideal $$(p,a)\triangleleft R$$. Since $$R$$ is a PID, we have $$(p,a)=(d)$$ for some $$d$$. Thus $$d|p$$ and $$d|a$$. Since $$d|p$$, $$p=q_1d$$ for some $$q_1$$. Since $$p$$ is irreducible, either $$q_1$$ or $$d$$ is a unit. If $$q_1$$ is a unit, then $$d=q_1^{-1}p$$ divides $$a$$. So $$a=q_1^{-1}px$$ for some $$x$$. This is a contradiction since $$p\nmid a$$. Thus $$d$$ is a unit. So $$(p,a)=(d)=R$$. Particularly, $$1_R\in (p,a)$$. Suppose $$1=rp+sa$$ for some $$r,s\in R$$. Then $$b=rpb+sab$$. Note that $$ab$$ is divisible by $$p$$, thus $$b$$ is divisible by $$p$$. So we're done. 

**Lemma.** Let $$R$$ be a principal ideal domain. Let $$I_1\subseteq I_2\subseteq I_3\subseteq\cdots$$ be a chain of ideals. Then there is some $$N\in\mathbb{N}$$ such that $$I_n=I_{n+1}$$ for some $$n>N$$.    
**Definition** (Ascending chain condition)**.** A ring satisfies the *ascending chain condition* (ACC) if there is no infinite strictly increasing chain of ideals.    
**Definition** (Noetherian ring)**.** A ring that satistifes the ACC is called a *Noetherian ring*.    
Every principal ideal domain in Noetherian.

### <center>Lecture 23. Gauss' Lemma, Eisenstein's Criterion</center>
**Lemma** (Gauss' lemma)**.**  Let $$R$$ be a UFD, and $$f\in R[X]$$ be a primitive polynomial. Then $$f$$ is reducible in $$R[X]$$ iff $$f$$ is reducible in $$F[X]$$, where $$F$$ is the field of fractions of $$R$$.

**Theorem.** If $$R$$ is a UFD, then $$R[X]$$ is a UFD.     
**Proposition** (Eisenstein's criterion)**.** Let $$R$$ be a UFD, and let

$$f=a_0+a_1X+\cdots+a_nX^n\in R[X]$$

be primitive with $$a_n\neq 0$$. If there exists irreducible $$p\in R$$ (hence prime) such that    
&emsp; 1. $$p\nmid a_n$$;    
&emsp; 2. $$p|a_i$$ for all $$0\leq i< n$$;      
&emsp; 3. $$p^2\nmid a_0$$.     
Then $$f$$ is irreducible in $$R[X]$$, and hence in $$F[X]$$.
